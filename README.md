# 🏠 우리 가족 사이트

가족을 위한 종합 웹사이트입니다. 로그인, 회원가입, 실시간 채팅, 채팅방, 가족 게임 등의 기능을 제공합니다.

## ✨ 주요 기능

### 🔐 인증 시스템
- 회원가입 및 로그인
- JWT 토큰 기반 인증
- 가족 코드를 통한 가족 그룹 관리

### 💬 실시간 채팅
- Socket.IO를 이용한 실시간 채팅
- 여러 채팅방 (가족 채팅, 부모님, 아이들)
- 메시지 히스토리 저장

### 🎮 가족 게임
- **가족 퀴즈**: 가족에 대한 재미있는 퀴즈
- **기억력 게임**: 카드 뒤집기 게임
- **그림 맞추기**: 캔버스에 그림을 그리고 맞추는 게임

### 📷 사진 갤러리
- 가족 사진 업로드 및 공유
- 반응형 갤러리 뷰

### 📅 가족 일정
- 달력 형태의 일정 관리
- 가족 이벤트 및 일정 추가

## 🛠 기술 스택

### Backend
- **Node.js** - 서버 런타임
- **Express.js** - 웹 프레임워크
- **Socket.IO** - 실시간 통신
- **MongoDB** - 데이터베이스
- **Mongoose** - MongoDB ODM
- **JWT** - 인증 토큰
- **bcryptjs** - 비밀번호 암호화

### Frontend
- **HTML5** - 마크업
- **CSS3** - 스타일링 (Grid, Flexbox, 애니메이션)
- **JavaScript (ES6+)** - 클라이언트 로직
- **Socket.IO Client** - 실시간 통신

## 📦 설치 및 실행

### 1. 프로젝트 클론 및 의존성 설치
```bash
# 의존성 설치
npm install
```

### 2. MongoDB 설정
MongoDB를 설치하고 실행하거나, MongoDB Atlas를 사용하세요.

로컬 MongoDB 실행:
```bash
mongod
```

### 3. 환경 변수 설정
`.env` 파일을 확인하고 필요에 따라 수정하세요:
```
MONGODB_URI=mongodb://localhost:27017/familysite
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
PORT=3000
```

### 4. 서버 실행
```bash
# 개발 모드 (nodemon 사용)
npm run dev

# 또는 일반 실행
npm start
```

### 5. 웹사이트 접속
브라우저에서 `http://localhost:3000`으로 접속하세요.

## 🚀 사용 방법

### 1. 회원가입
- 사용자명, 이메일, 비밀번호, 가족 코드를 입력하여 회원가입
- 같은 가족 코드를 사용하는 사용자들이 하나의 가족 그룹을 형성

### 2. 로그인
- 이메일과 비밀번호로 로그인
- 로그인 후 자동으로 메인 화면으로 이동

### 3. 채팅 사용
- 좌측에서 채팅방 선택 (가족 채팅, 부모님, 아이들)
- 메시지 입력 후 전송 버튼 클릭 또는 Enter 키
- 실시간으로 다른 가족 구성원과 채팅 가능

### 4. 게임 즐기기
- 게임 탭에서 원하는 게임 선택
- 각 게임의 규칙에 따라 플레이
- 가족과 함께 즐길 수 있는 다양한 게임

### 5. 사진 공유
- 사진 탭에서 가족 사진 업로드
- 업로드된 사진들을 갤러리 형태로 확인

### 6. 일정 관리
- 일정 탭에서 가족 일정 확인
- 새로운 일정 추가 및 관리
